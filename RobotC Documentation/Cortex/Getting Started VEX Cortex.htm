Getting Started with ROBOTC

The VEX Cortex is a fully programmable device, and is what enables you to incorporate motors, sensors, an LCD screen, and remote control signals all in one robot. Inside of the Cortex, there are two separate processors; a user processor handles all of the ROBOTC programming instructions, and a master processor controls lower-level operations, like motor control and VEXnet communication.


The VEX Cortex can be connected directly to the computer using the USB A-to-A cable, or wirelessly using a VEXnet connection. Many of the configuration steps are the same whether you are programming using a wired or wireless connection, but there are some important differences.
